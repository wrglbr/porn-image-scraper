## porn-image-scraper

### Currently supported sites:

- porn-star.com

- tubsexer.com
- coedcherry.com
- pornpics.com
- sweet-pornstars.com
- europornstar.com
- **_NEW_** _imagefap.com_
### How to

You'll need the node.js runtime (https://nodejs.org/en/)

```
npm install
node dist url0 url1 [...urls]
```

With each url being 1 gallery.
Images will be stored in ./images/ folder by default.
